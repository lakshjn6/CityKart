**NiwaiKart** is a comprehensive e-commerce platform designed to bridge the gap between local city commerce and digital convenience. 
Built with a powerful Django backend and a responsive frontend, it provides a seamless shopping experience for users and
a full-featured management suite for administrators.

## üöÄ Key Features

### **User Experience**
* **Secure Authentication:** User registration, login, and profile management using Django's robust authorization system.
* **Dynamic Product Catalog:** Browse products by category with real-time availability updates.
* **Persistent Shopping Cart:** Seamlessly add, update, or remove items from the cart.
* **Payment Gateway Integration:** Secure checkout process integrated with modern payment APIs.
* **Order History:** Users can track their past purchases and current order status.

### **Admin Capabilities**
* **Full Admin Panel:** A customized dashboard to manage the entire store.
* **Inventory Control:** Admins can add new products, update descriptions, and manage images.
* **Quantity Management:** Real-time tracking of stock levels and product quantities.
* **Transaction Oversight:** Monitor successful payments and pending orders.
